# Code of Conduct

Be useful. Be direct. Be kind enough to be clear.

This project is about making agents better collaborators. That applies to humans too.

## Expected

- Discuss ideas clearly.
- Critique workflows, not people.
- Prefer concrete examples.
- Keep bilingual contributions readable.

## Not Expected

- Personal attacks
- Credential sharing
- Prompt spam
- Vague “make it better” advice with no actionable detail

If something feels off, open an issue with the smallest concrete example possible.

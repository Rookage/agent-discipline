# Skills

This directory contains the five Agent Discipline skills.

| Skill | Purpose |
|---|---|
| `clarify-first` | Clarify before building |
| `repro-first` | Reproduce before fixing |
| `behavior-first` | Test behavior before refactoring |
| `session-handoff` | Preserve context between sessions |
| `fast-operator` | Execute quickly without making the user operate the agent |

Each skill is intentionally small. Agent discipline should feel like a sharp tool, not a meeting agenda.
